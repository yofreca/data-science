# Plan de Trabajo para Tutorial Animado de Ciencia de Datos

## 1. Objetivo General
Diseñar y desarrollar un tutorial web animado, altamente interactivo, que enseñe los fundamentos necesarios para convertirse en científico de datos, incluyendo matemáticas, estadística, programación, herramientas de software, buenas prácticas y construcción de portafolio.

## 2. Público Objetivo
- Estudiantes con interés en ciencia de datos.
- Profesionales que buscan reconversión laboral.
- Personas con conocimientos básicos de matemáticas y programación.

## 3. Estructura General del Tutorial
El tutorial estará dividido en **módulos temáticos**, cada uno con:
- Animaciones explicativas.
- Ejemplos interactivos.
- Pruebas rápidas.
- Pequeñas actividades prácticas.
- Descargables o snippets de código.

---

## 4. Módulos del Tutorial

### **Módulo 1. Introducción a la Ciencia de Datos**
- ¿Qué es la ciencia de datos?
- Roles relacionados: Data Analyst, Data Engineer, ML Engineer, Researcher.
- Ciclo de vida de un proyecto de ciencia de datos.
- Herramientas principales (Python, R, SQL, Git, plataformas cloud).
- Mini-actividad: identificar etapas del ciclo en un caso real animado.

---

### **Módulo 2. Fundamentos Matemáticos**
#### **2.1 Álgebra Lineal**
- Vectores, matrices, operaciones.
- Transformaciones lineales.
- Descomposiciones (SVD, PCA como aplicación animada).
- Actividad visual: mover sliders para ver efectos en transformaciones.

#### **2.2 Cálculo**
- Límites y derivadas.
- Gradiente, optimización.
- Aplicación visual: descenso del gradiente animado.

#### **2.3 Probabilidad**
- Regla de la suma y producto.
- Teorema de Bayes animado.
- Distribuciones (normal, binomial, Poisson).
- Actividad: simulador de probabilidades.

---

### **Módulo 3. Estadística Fundamental**
- Medidas estadísticas.
- Muestreo y estimación.
- Intervalos de confianza.
- Pruebas de hipótesis.
- Actividad: experimento de monedas con simulador.

---

### **Módulo 4. Programación para Ciencia de Datos**
#### **4.1 Python**
- Sintaxis base.
- Estructuras de datos.
- Librerías esenciales (NumPy, Pandas, Matplotlib, Scikit-Learn).
- Mini ejercicios animados.

#### **4.2 SQL**
- Select, joins, agregaciones.
- Modelos relacionales.
- Actividad: práctica interactiva con base de datos simulada.

#### **4.3 Versionamiento con Git**
- Commits, ramas, merges.
- Flujo Git animado.

---

### **Módulo 5. Limpieza y Manipulación de Datos**
- Tipos de datos.
- Manejo de valores faltantes.
- Outliers.
- Normalización y estandarización.
- Actividad: transformar datasets paso a paso.

---

### **Módulo 6. Visualización de Datos**
- Principios básicos (percepción, colores, escalas).
- Librerías (Matplotlib, Seaborn, Plotly).
- Actividad: crear gráficos interactivos.

---

### **Módulo 7. Machine Learning Clásico**
#### **7.1 Supervisado**
- Regresión lineal, logística.
- Árboles de decisión, Random Forest.
- SVM.
- Pipeline visual animado.

#### **7.2 No Supervisado**
- K-means.
- PCA.
- Clustering.
- Actividad: agrupar puntos moviendo parámetros.

#### **7.3 Validación de Modelos**
- Overfitting y underfitting.
- Cross-validation.
- Métricas.
- Actividad: evaluar modelos con widgets.

---

### **Módulo 8. Deep Learning (Opcional Avanzado)**
- Redes neuronales.
- Perceptrón.
- Backpropagation animado.
- CNN y RNN introducción.
- Mini simulador de una red simple.

---

### **Módulo 9. Desarrollo de Proyectos Reales**
- Cómo estructurar un proyecto.
- EDA.
- Elección del modelo.
- Entrenamiento y validación.
- Documentación.
- Uso de notebooks.
- Actividad: mini-proyecto guiado.

---

### **Módulo 10. Herramientas de Software Modernas**
- GitHub.
- Docker.
- Entornos virtuales (venv, conda).
- Plataformas cloud (Azure, AWS, GCP).
- MLOps básico.

---

### **Módulo 11. Buenas Prácticas y Ética en Ciencia de Datos**
- Sesgos.
- Privacidad.
- Interpretabilidad.
- Documentación.
- Actividad: caso de estudio.

---

### **Módulo 12. Construcción de Portafolio**
- Proyectos recomendados.
- Cómo escribir README profesionales.
- Deploy de modelos.
- Ejemplos de portafolios.

---

## 5. Tecnologías Recomendadas para el Tutorial Web
- **Frontend:** React, Svelte o Next.js.
- **Animaciones:** GSAP, Lottie, Framer Motion.
- **Interactividad:** Canvas, WebGL, D3.js.
- **Backend (si aplica):** Node.js / Python FastAPI.
- **Infraestructura:** GitHub Pages, Vercel, Netlify.
